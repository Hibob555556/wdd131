Site Purpose

The purpose of this site is to consolidate many of the tools which Test Engineers need into one convenient location.

Scenarios

What is Test Buddy?  ▼

Test Buddy is designed for Test Engineers who need to quickly access a wide variety of tools. Test Buddy combines tools for string comparison and generation and binary conversions in one place.

Why did you make Test Buddy?  ▼

I made Test Buddy because as a Test Engineer I have had issues finding a lot of the common tools which I find myself needing in one convenient location.

What is the string comparator for?  ▼

The String Comparator performs a basic string comparison and returns a simple boolean result (true/false).
